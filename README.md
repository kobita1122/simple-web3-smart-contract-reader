# Simple Web3 Smart Contract Reader

This is a beginner-friendly blockchain project.

This project helps you:
- Connect a crypto wallet
- Read blockchain network
- Read public data from a smart contract
- Display contract values on screen

This project uses:
- HTML
- JavaScript
- ethers.js
- Smart contract ABI

## Features
- One-click wallet connect
- Read contract data
- Show network name
- Simple input field for contract address

## Requirements
- MetaMask installed
- Internet connection
- Basic JavaScript knowledge

## How to Run
1. Download this repository
2. Open `index.html` in browser
3. Click "Connect Wallet"
4. Enter contract address
5. Click "Read Contract"
6. See result on screen

## File Structure
- index.html -> Main webpage
- app.js -> App logic
- config.js -> App settings
- provider.js -> Web3 provider
- connect.js -> Wallet connect
- network.js -> Network reader
- contract.js -> Contract logic
- ui.js -> Screen updates
- abi.js -> Contract ABI
- utils.js -> Helper functions

## Learning Goal
This project helps beginners learn:
- How smart contracts work
- How to read blockchain data
- How ABI connects frontend to contract
